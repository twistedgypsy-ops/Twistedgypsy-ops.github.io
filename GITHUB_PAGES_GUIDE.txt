
GITHUB PAGES — QUICK STEPS

1) Create repo named yourusername.github.io (public).
2) Upload all files to the repo root.
3) Settings → Pages → Branch: main, Folder: /(root). Save.
4) Visit https://yourusername.github.io in ~1 minute.

Shop links: edit shop.html
Mailchimp: replace YOUR_MAILCHIMP_POST_URL in footer
Contact form: replace Formspree URL in contact.html
